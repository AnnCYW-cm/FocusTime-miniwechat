# 图标资源说明

本项目需要以下图标文件，请自行准备或使用设计工具创建：

## Tab Bar 图标

需要在 `images/` 目录下准备以下图标文件：

### 首页图标
- `tab-home.png` - 未选中状态（灰色，81x81px）
- `tab-home-active.png` - 选中状态（番茄红 #E74C3C，81x81px）

### 历史图标
- `tab-history.png` - 未选中状态（灰色，81x81px）
- `tab-history-active.png` - 选中状态（番茄红 #E74C3C，81x81px）

### 统计图标
- `tab-stats.png` - 未选中状态（灰色，81x81px）
- `tab-stats-active.png` - 选中状态（番茄红 #E74C3C，81x81px）

### 我的图标
- `tab-profile.png` - 未选中状态（灰色，81x81px）
- `tab-profile-active.png` - 选中状态（番茄红 #E74C3C，81x81px）

## 其他图标

### 默认头像
- `default-avatar.png` - 用户默认头像（200x200px）

## 图标设计建议

1. **尺寸**：81x81px（@1x），建议提供 @2x 和 @3x 版本
2. **格式**：PNG 格式，支持透明背景
3. **风格**：简约扁平化设计，线条清晰
4. **颜色**：
   - 未选中：#7F8C8D（灰色）
   - 选中：#E74C3C（番茄红）

## 临时替代方案

如果暂时没有图标，可以：
1. 使用在线图标库（如 iconfont.cn）下载合适的图标
2. 使用 Figma、Sketch 等设计工具自己绘制
3. 在开发阶段先使用纯色占位图，后续再替换

## 推荐图标内容

- **首页**：番茄图标或房子图标
- **历史**：时钟或历史图标
- **统计**：柱状图或折线图图标
- **我的**：人像或用户图标
